# Optimisation des salles de cours - EMSI Marrakech

Projet de Programmation Linéaire et Python.

## Objectif

Optimiser la répartition des heures de cours entre deux filières en utilisant la programmation linéaire.

## Technologies utilisées

- Python
- PuLP
- Matplotlib

## Structure du projet

- src/ : code source
- images/ : graphiques générés
- resultats/ : résultats exportés
- rapport/ : rapport final

## Exécution

Installer les dépendances :

```bash
pip install -r requirements.txt
```

Lancer le programme :

```bash
python src/main.py
```
